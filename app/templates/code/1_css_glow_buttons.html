Enhance Button Hover Effects with CSS Text Shadow

In many places across the web, I see a flashy effect used to enhance the appeal of the sites links and buttons. For the most part, it's always implemented with images. However, all the major browsers( except IE) support CSS attributes that can achieve this same effect. And for IE, you can either fall back to images, use their proprietary hacks, or fall back to a simple supported standard.

Mouse Over Me

The subtle glow around the letters on hovering gives a nice gentle feel that lets the user know they've found an action item.

Let's take a look at the code:

		      
<style type="text/css">

  #button-demo{ padding: 1em}

  #button-demo a{ background: #666666;}

  #button-demo a{ color: #FFF }

  #button-demo a:hover{ text-shadow: 1px 1px 10px #FFF;

                        text-decoration:none}

</style>

<!--[if IE]> 

  <style type="text/css">

    #button-demo a:hover{text-decoration:underline}

  </style>

<![endif]-->

<div id="button-demo">

  <a href="#">Mouse Over Me</a>

</div>

My solution simply uses underline for ie support. It's an elegant solution from the code's perspective and it still gives users of that browser the usability advantages of the hover effect. As mentioned above, with a little labor, a solution can be made to support ie in the same manner as standard css is supported by Firefox, Chrome, Safari, etc.

Combining this effect with a slight change in text color can yield great results.
